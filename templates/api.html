API Instructions


    Collection Endpoint

    To get a collection json attach: /movies/categories/bestpicture/1997 at the end of the url.
    This url will return a json collection of all the oscar movies that have won or were nominated best picture in 1997.
    Example: http://website.com/movies/categories/bestpicture/1997


    Singleton Enpoint

    To get a singleton json attach: /movies/categories/bestpicture/1997/winner at the end of the url. This url will return a json singleton for the oscar movie that won best picture in 1997.
    Example: http://website.com/movies/categories/bestpicture/1997/winner

    Search Enpoint

    To search for movies through the url, use the format http://website.com?category=[CATEGORY]&winner=[WINNER]&name=[NAME]&year=[YEAR].
    replace [CATEGORY] with the category you want to search for. If you want to exclude a filter, for exmple the winner filter, remove &winner=[WINNER] from the url.
    Example: http://website.com/search?category=direction&winner=True&year=1987
    Example: http://website.com/search?category=direction&winner=True
    Example: http://website.com/search?name=Iron
